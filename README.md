# 3D Avatar Viewer (Flutter + o3d)
A lightweight Flutter app that loads and shows a 3D avatar using the o3d package (GLB/GLTF support), with basic camera controls (rotate / zoom) and local model loading.
This project loads the avatar directly from a URL generated by the Ready Player Me website â€” no assets folder or local model files required.

# Features
  Display 3D avatar using the o3d package
  Load model from remote URL
  Supports rotation, zoom, and panning
  Clean, minimal codebase
  Perfect as a starter template for 3D model viewers

# Usage
  Replace the src value in the viewer widget with your own avatar URL:
  O3DViewer(
    src: 'https://models.readyplayer.me/690cbf7fd28f9f8e507ec593.glb',
    enableRotate: true,
    enableZoom: true,
    enablePan: true,
  )
